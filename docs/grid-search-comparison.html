Grid Search Parameters ? What is this?

Entry/Exit: Threshold ranges for probability divergence. Fees: Kalshi trading fees in P&L. Slippage: Price movement cost. Min Trades: Minimum trades for valid combo. Split: Train/valid/test ratios. Exclude: Seconds excluded from game start/end.

All models were tested using the same grid search parameters.
