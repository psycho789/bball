Summary Comparison ? Summary Comparison Metrics
  • Test Profit: Total net profit after fees. Primary performance metric.
  • Profit Imp.: Percentage change vs ESPN baseline. Positive = better than baseline.
  • Profit/Trade: Average profit per trade. Higher = more efficient use of opportunities.
  • P/T Imp.: Profit/trade improvement vs baseline. Shows efficiency gains.
  • Trades: Number of trades executed. More trades = more opportunities but also more fees.
  • Win Rate: Percentage of winning trades. Higher = more reliable predictions.
  • Profit Factor: Gross profit / Gross loss. Higher = better risk-adjusted returns (industry standard).
  • Max DD: Maximum drawdown (peak-to-trough decline). Lower = less risk, need less capital buffer.
  • Entry/Exit: Optimal threshold values chosen by grid search.

Grid Search Parameters ? What is this?

Entry/Exit: Threshold ranges for probability divergence. Fees: Kalshi trading fees in P&L. Slippage: Price movement cost. Min Trades: Minimum trades for valid combo. Split: Train/valid/test ratios. Exclude: Seconds excluded from game start/end.

All models were tested using the same grid search parameters.

Entry: 0.02 - 0.2 (step 0.01)
Exit: 0 - 0.05 (step 0.005)
Fees: Yes
Slippage: 0
Min Trades: 200
Split: 0.7/0.15/0.15
Exclude: 60s/60s
